IT Programmer Applicant Hands On Examination

  • Download the following:
    • Microsoft Visual Studio 2015
    • MySQL Community Server
  • Create a new database and name it db_test in MySQL Community Server
  • Create the following tables in db_test:
    • user_account
      • id: primary key, bigint
      • username: varchar(30)
      • password: varchar(250)
      • first_name: varchar(250)
      • middle_name: varchar(250)
      • last_name: varchar(250)
      • is_active: tinyint
      • department_id
    • department
      • id: primary key, bigint
      • name: varchar(250)
  • Create a Visual Basic Windows Forms Application that will serve as UI for db_test in Visual Studio 2015. Name the project with your name.
    • Project Name Format: [Last Name]_[First Name]
  • Your UI should include the following:
    • Login form
    • Main menu
    • List of user accounts form
    • Create/update user account form
    • List of departments form
    • Create/update department form
  • Publish your VB Project to GitHub and send us the link
  • OPTIONAL: User account password encryption
GOOD LUCK!
